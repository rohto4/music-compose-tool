# Project Agent Routing

## 現在利用する手順

- `frontend-design-direction`: 目的、hierarchy、spatial model、visual system、signature detail、responsive制約を明示する。
- `brand-discovery`: 長期の質問をmodule化し、原則1回に1問、回答とsynthesisを保存する。
- Hallmark 1.1.0: ユーザー指定の外部設計source。出所と採用境界は `docs/candi-ref/hallmark-adoption.md`。

## 非採択

- AI skill母集団の一括copy。
- framework/language専用skill。stack未決のため。
- deployment、cloud、connector、automation関連設定。Phase 0の対象外のため。

Hallmarkは現時点でproject skillとしてinstallせず、pinしたread-only sourceを参照する。production designを開始する時点で、必要なreferenceだけを読み、licenseとversionを再確認する。
